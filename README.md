## 📁 디렉토리
모든 노트북 파일(.ipynb)는 각 모델을 정의, 훈련, 검증, 평가, 제출물 제작 과정을 담음.
models 디렉토리 안에는 각 모델의 훈련 과정중 파라미터를 저장함.
```
.
├── models
│   ├── densenet121
│   ├── mobilenet_v3_small
│   ├── resnet18
│   └── resnet50
├── README.md
├── densenet.ipynb
├── mobilenet.ipynb
├── resnet18.ipynb
└── resnet50.ipynb
```
